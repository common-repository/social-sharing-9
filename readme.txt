This is a customizable Social Sharing plugin for WordPress.

== Description ==

Social Sharing 9 is a customizable WordPress plugin that enables you to display social sharing icons in different places on your website.

It enables an admin page where you can select all your options, choose what social networks you'll activate, choose sizes, colors ad even order them as you wish.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/social-sharing-9` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the `Social Sharing 9` screen to configure the plugin's options


== Frequently Asked Questions ==

= Why isn't the Whatsapp button being displayed? =
